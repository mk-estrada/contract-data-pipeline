# contract-data-pipeline
Automated contract analytics pipeline for executive decision support (Python, SQL, Power BI)

## ⚠️ Non-runnable public version (by design)

This repository is a **sanitized code reference** of an internal pipeline I built.
It is **not runnable** in this public form because proprietary endpoints, credentials,
authentication flows (including MFA), and internal data have been intentionally removed.
The purpose is to demonstrate pipeline structure, transformation logic, and data-quality patterns.
